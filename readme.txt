An immediate way to extract a winner from your giveaway posts!

== Description ==

Create a post and tag it "giveaway".

Ask readers to leave a comment to win a fantastic prize.

Wait enough time. Then close the contest.

Open the Giveaway plug-in panel, choose the post and extract the winner.

That's all. Ultra lite, ultra easy, no hassle at all.

Official page (http://www.satollo.net/plugins/giveaway).


== Installation ==

1. Put the plug-in folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Optional: go to the options page and configure the plugin
